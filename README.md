Sales Data Cleaning & Analysis Excel Project

**Project Overview
This project focuses on cleaning, standardizing, and preparing raw sales data using Microsoft Excel for data analysis and visualization.

**Data Cleaning Steps
- Removed unwanted special characters from data
- Cleaned inconsistent product category values
- Standardized region names
- Removed extra spaces and formatting issues
- Corrected inconsistent text values
- Prepared clean data for analysis

**Tools Used
- Microsoft Excel
- Power BI

** Excel Functions Used
- TRIM()
- SUBSTITUTE()
- PROPER()
- IF()
- SEARCH()
- UPPER()

** Data Transformation
- Converted messy raw data into structured format
- Created clean category and region columns
- Improved data quality for reporting

** Dashboard Analysis
Created sales insights dashboard including:
- Total Sales
- Total Profit
- Product Category Analysis
- Region-wise Performance
- Sales & Profit Insights

## Result
Successfully transformed raw and inconsistent sales data into a clean, structured dataset ready for analysis and business reporting.
